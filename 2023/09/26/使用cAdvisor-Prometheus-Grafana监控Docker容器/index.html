¶2.0 cAdvisor 的部署

¶4.0 Grafana 的部署

  1. 因为之前已经把所有的镜像都已经下载了，所以直接用 run 来运行就行了，之后在浏览器中使用 你的IP:25573/login 来访问即可

  1. 默认账号及密码都为 admin ，登录之后网站会强制你更新密码
