记一次电脑启动原神蓝屏解决方案

发表于 2023-05-06- 更新于 2023-09-05
- -
编程技巧-原创计算机

  • 原因：

    电脑启动原神时蓝屏，退出代码为 kernel-security_check_failure（内核安全检查失败）

  • 解决方法：

    进入设置 windows 安全中心将内核隔离的内核模式硬件强制堆栈保护关闭即可
